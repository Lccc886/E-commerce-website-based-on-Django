# 快速使用提示词

## 一键生成（推荐）

直接复制以下提示词到 Claude Code：

```
请为当前Django项目生成完整的项目文档图表，包括：
1. 数据流图（详细版）- 展示用户、管理员与系统的数据交互
2. 数据库E-R图（简洁版）- 展示核心实体关系
3. 功能模块图 - 分别展示用户端和管理员端功能
4. 用例图 - 分别展示用户端和管理员端用例

要求：
- 分析项目的models.py和views.py文件
- 使用Mermaid语法
- 保存到桌面的"项目图表"文件夹
- 图表内容使用中文
```

## 分项生成

### 只生成ER图

```
请分析项目的所有models.py文件，生成数据库E-R图。
使用Mermaid erDiagram语法，显示所有实体、字段和关系。
保存到桌面的"项目图表/数据库ER图.md"
```

### 只生成数据流图

```
请分析项目的业务逻辑，生成详细的数据流图。
包含顶层数据流图和第一层数据流图。
使用Mermaid flowchart语法，保存到桌面的"项目图表/数据流图.md"
```

### 只生成功能模块图

```
请分析项目的views.py文件，生成功能模块图。
分别展示用户端和管理员端的功能模块。
使用Mermaid mindmap语法，保存到桌面的"项目图表/功能模块图.md"
```

### 只生成用例图

```
请分析项目的功能需求，生成用例图。
分别展示用户端和管理员端的用例，包含用例说明表。
使用Mermaid语法，保存到桌面的"项目图表/用例图.md"
```

## 更新图表

当项目代码变更后，使用以下提示词更新图表：

```
项目代码已更新，请重新生成所有项目图表，覆盖原有文件。
保存到桌面的"项目图表"文件夹。
```
