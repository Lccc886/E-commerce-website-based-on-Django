# LUMINA 商城 - 基于 Django 的 B2C 电商平台

## 项目简介

LUMINA 商城是一个基于 Django 框架开发的轻量级 B2C 电商平台，旨在为中小型企业和个人创业者提供一个功能完整、易于维护、界面现代的在线销售解决方案。项目涵盖了用户认证、商品管理、购物车、订单结算、优惠券、商品评价等核心电商功能，并采用响应式布局，适配 PC 端、平板和手机访问。

## 技术栈

| 类别 | 技术 |
|------|------|
| 后端框架 | Django 4.x |
| 数据库 | MySQL 8.0 |
| 缓存/验证码 | Redis 7.0 |
| 前端样式 | Tailwind CSS + Bootstrap 5 |
| 前端交互 | 原生 JavaScript (Ajax) |
| 邮件服务 | QQ 邮箱 SMTP |
| 部署环境 | Nginx + Gunicorn (生产) |

## 主要功能

### 用户端
- **注册与登录**：支持邮箱验证码注册，用户名/密码登录；表单实时校验（长度、格式、密码强度）
- **商品浏览**：分类筛选、关键词搜索、分页展示；商品详情页展示价格、库存、描述及用户评价
- **购物车**：无刷新添加商品（Ajax），支持数量修改、删除；购物车图标实时更新总数
- **订单结算**：多地址管理（新增/编辑/删除/设为默认）、优惠券验证、库存事务扣减
- **个人中心**：订单列表（按状态分组）、收货地址管理、订单详情查看
- **商品评价**：已购买用户可打分（1-5星）和撰写评论，商品详情页展示平均分及评论列表

### 管理端（Django Admin 定制）
- 商品管理（增删改查、上下架、库存修改）
- 订单管理（状态流转、批量操作）
- 用户管理（禁用/启用）
- 优惠券管理（创建、有效期、使用限制）
- 评价审核（通过/删除）

## 项目特色

- **无刷新购物体验**：采用 Ajax 异步请求加入购物车，页面不刷新，配合 Toast 弹窗提示，操作流畅。
- **高并发库存安全**：订单创建时使用数据库事务 + `select_for_update()` 行级锁，防止超卖；超时未支付订单自动取消并恢复库存。
- **Redis 缓存优化**：验证码存储于 Redis（5分钟过期），商品分类菜单缓存 1 小时，减少数据库压力。
- **响应式界面**：基于 Tailwind CSS 断点设计，在手机、平板、PC 上布局自适应；移动端导航栏折叠为汉堡菜单。
- **安全防护**：CSRF 中间件保护所有 POST 请求；Django ORM 参数化查询防 SQL 注入；模板自动转义防 XSS；密码 PBKDF2 哈希存储。
- **邮箱验证码**：注册流程强制邮箱验证，验证码通过 QQ 邮箱 SMTP 发送，防止恶意注册。

## 部分代码结构

项目按功能拆分为多个 Django 应用：

```
lumina_store/
├── main/          # 首页、基础模板
├── goods/         # 商品、分类、评价
├── cart/          # 购物车（基于 session）
├── orders/        # 订单、结算
├── users/         # 用户认证、地址管理
├── coupons/       # 优惠券（后台管理）
└── templates/     # 全局模板
```

## 运行与部署

### 开发环境
```bash
git clone <repository>
cd lumina_store
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### 生产环境（简要）
- 使用 Nginx 反向代理 + Gunicorn
- 配置 MySQL 和 Redis
- 设置环境变量（SECRET_KEY, 数据库密码, 邮箱授权码等）
- 定时任务：`python manage.py release_expired_orders`（释放超时订单）

## 总结

LUMINA 商城是一个功能完整、代码清晰、易于二次开发的 Django 电商项目。它既能作为毕业设计的参考实现，也可作为中小商家搭建自营电商平台的起点。后续可扩展支付集成、多规格 SKU、秒杀活动等高级功能。
个人项目，不参与商业开发,另外存在前后端分离版本，基于Django+Vue3开发
